# Portafolio Web para Camionero

Este repositorio contiene un ejemplo sencillo de un sitio web de portafolio para un camionero. Está construido únicamente con **HTML** y **CSS** e incluye las siguientes secciones:

- Inicio con imagen de fondo y presentación.
- Sobre mí con una breve biografía y experiencia.
- Servicios de transporte ofrecidos.
- Galería con imágenes ilustrativas.
- Formulario de contacto y datos de localización.

Para visualizar el sitio, abre `index.html` en tu navegador.
